# Roadmap

## Phase 1: MVP Dashboard

- Build dashboard overview, product catalog, sales checkout, customers, suppliers, subscriptions, and reports.
- Add sample data and interactive browser behavior.
- Create PRD and roadmap documentation.

## Phase 2: Backend Foundation

- Add Node.js API server.
- Add PostgreSQL schema for users, businesses, products, sales, payments, subscriptions, and roles.
- Add JWT authentication and role-based access control.
- Add transactional sales endpoint that updates inventory atomically.

## Phase 3: Billing and Multi-Tenancy

- Add Paystack and Flutterwave subscription billing.
- Add tenant isolation by business ID.
- Add plan limits for users, products, branches, and report access.
- Add free trial and upgrade/downgrade flows.

## Phase 4: Production Readiness

- Add Redis caching.
- Add audit logs.
- Add exportable PDF and Excel reports.
- Add backups, monitoring, and uptime checks.
- Add offline mode with sync.

## Phase 5: Growth Features

- Mobile POS app.
- Barcode scanning.
- AI sales forecasting.
- Accounting integrations.
- Multi-currency support.
