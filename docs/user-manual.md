# User Manual

## Business Owner

1. Visit the home page.
2. Register with business details.
3. Select a subscription plan.
4. Log in to the dashboard.
5. Add products and staff.
6. Use reports to monitor performance.

## Cashier

1. Log in with cashier credentials.
2. Open the POS page.
3. Search for products.
4. Add products to the cart.
5. Choose payment method.
6. Complete sale and print receipt.

## Super Admin

1. Log in with Super Admin credentials.
2. View platform dashboard.
3. Monitor businesses, subscriptions, payments, and audit logs.
4. Activate or suspend businesses when needed.
