# Project Overview

The project is a cloud-based subscription POS system for small and medium enterprises. It helps businesses manage sales, inventory, customers, suppliers, expenses, reports, staff, and subscriptions from a single web application.

The system uses a SaaS model so businesses can start with a free trial, upgrade to paid plans, and access features according to their subscription level.

## Main Users

- Super Admin / Platform Owner
- Business Owner / SME Admin
- Cashier / Staff

## Main Modules

- Public website
- Authentication and business registration
- Subscription and payment management
- POS checkout
- Products and inventory
- Sales and receipts
- Customers and suppliers
- Expenses and reports
- Staff and permissions
- Settings, notifications, and audit logs
- Super Admin platform management
